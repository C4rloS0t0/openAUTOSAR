This "common" folder comes from the open source code from Arctic.
Although some of the files in it have been moved to src subfolder 
after the fork, there are possibly still files that 
could be needed in the future.
In order to be able to recompile the system and provide 
a development envirobnment as soon as possible it has been decided to
consider the handling of the files in this folder as a work in progress,
also in the light of new hardware support.
